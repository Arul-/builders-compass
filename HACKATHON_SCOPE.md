- Photon framework
- Beam UI runtime
- Photon marketplace tooling
- TinyFish platform and API

## Built during the hackathon

- `builder-compass.photon.ts`
- The Builder Compass product concept and builder-persona framing
- Persistent builder profile flow
- Identity analysis and market-path scoring logic
- TinyFish-powered market research integration
- Custom `ui/builder-compass.html` dashboard
- Declarative `ui/builder-compass.photon.html` dashboard
- Marketplace packaging for standalone install and testing

## What Builder Compass does

Builder Compass helps AI-native builders understand what kind of builder they are, how similar profiles are making money on the open web, and what their fastest and strongest next paths are.

Users can:

- save a persistent builder profile
- bring their own TinyFish API key
- run identity analysis
- run market-path research
- reuse saved results without re-running TinyFish every time
- deliberately rebuild the profile when their direction changes
